---
title: Upgrading to Gulp 4 via NPM
date: '2018-07-22'
---

```bash
$ npm uninstall --global gulp gulp-cli
# Uninstalls gulp (and gulp-cli if you have it) globally
$ npm install --global gulp@next gulp-cli@next
# Installs the beta (version 4.0.0) release of gulp and the latest release of gulp-cli
```
